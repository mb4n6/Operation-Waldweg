## Scenario

A missing person case escalates into a suspected homicide in a rural area near Stuttgart.  
Digital traces from multiple sources — including smartphones, cloud data, communication logs,  
health data, location tracking, and multimedia files — must be analysed to reconstruct events.

**Case:** AZ 2026-KK-00892 · **Period:** 24 Jan – 25 Jan 2026 · **Status:** unresolved (fictional)
